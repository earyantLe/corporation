沟通准则

  • ✓ 及时响应：工作消息 2 小时内回复，紧急事项立即响应
  • ✓ 透明沟通：信息共享，避免信息孤岛
  • ✓ 尊重他人：倾听不同意见，建设性反馈
  • ✗ 避免：背后议论、推诿责任、情绪化沟通
🎯

决策准则

  • ✓ 数据驱动：基于数据和事实做决策
  • ✓ 快速迭代：小步快跑，快速验证，及时调整
  • ✓ 权责对等：谁决策谁负责，明确责任主体
  • ✗ 避免：拍脑袋决策、拖延不决、责任不清
⚡

执行准则

  • ✓ 结果导向：以终为始，确保目标达成
  • ✓ 主动推进：不等不靠，主动解决问题
  • ✓ 质量第一：不妥协于短期压力，保证质量
  • ✗ 避免：敷衍了事、拖延症、降低标准
🤝

协作准则

  • ✓ 边界清晰：明确职责边界，避免推诿
  • ✓ 补位意识：主动补位，不袖手旁观
  • ✓ 知识分享：乐于分享，共同成长
  • ✗ 避免：部门墙、本位主义、信息垄断
