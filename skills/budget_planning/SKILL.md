# 💰 预算规划 · Budget Planning

> CFO 专属技能：预算制定与控制

---

## 技能描述

本技能用于制定项目预算，跟踪预算执行情况，进行成本效益分析。

**使用者**: CFO (主)、财务 (辅)

**适用场景**:
- 项目预算制定
- 预算执行跟踪
- 成本效益分析
- 预算调整建议

---

## 使用方法

### 命令格式

```markdown
💰 预算规划

**项目名称**: {项目名称}
**预算类型**: {项目预算/年度预算/临时预算}
**预算周期**: {时间范围}
```

### 输出格式

```markdown
💰 预算报告

**项目**: {项目名称}
**预算周期**: {时间范围}

**预算明细**:
| 类别 | 金额 | 占比 | 说明 |
|------|------|------|------|
| {类别} | ¥{金额} | {比例} | {说明} |

**总计**: ¥{总金额}

**财务建议**:
{建议内容}
```

## 预算管理规范

### 预算制定原则

1. **实事求是**：根据实际需求制定
2. **留有余地**：设置适当的预备金
3. **精打细算**：能节约的尽量节约
4. **重点保障**：优先保障核心需求

### 预算执行原则

1. **按预算执行**：不超预算开支
2. **及时记账**：当日开支当日记账
3. **定期分析**：每周分析执行情况
4. **调整审批**：预算调整需审批

---

## 注意事项

1. 预算制定要合理，不能过高或过低
2. 预算执行要跟踪，及时发现偏差
3. 成本效益分析要客观，数据要准确
4. 预算调整要谨慎，需 CEO 审批
5. 预算报告要定期发送给 CEO/COO
