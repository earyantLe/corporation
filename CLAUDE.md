# CLAUDE.md

## 项目概述

**Corporation** - OpenClaw 框架的 AI 多 Agent 协作技能包，包含 12 个公司角色和 5 个核心技能。

## 文件结构

- `agents/{role}/SOUL.md` - Agent 人格定义
- `skills/{skill}/SKILL.md` - 技能定义

## 常用命令

```bash
./install.sh                  # 安装
openclaw gateway start        # 启动 Gateway
openclaw agent --agent ceo --message "..."  # 测试 Agent
```
