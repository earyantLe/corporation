## 系统概述

本系统是一个基于 AI 多 Agent 协作的企业级管理系统，包含 12 个核心部门角色，模拟真实公司的组织架构和运营流程。

### 12 个 Agent 角色

```
CEO (首席执行官)
├── COO (首席运营官)
├── CFO (首席财务官)
└── CTO (首席技术官)
    ├── 工程 (Engineering)
    ├── 设计 (Design)
    └── QA (质量检测)

独立部门:
├── 人力 (HR)
├── 财务 (Finance)
├── 法务 (Legal)
├── 市场 (Marketing)
└── 销售 (Sales)
```

## 核心功能模块

### 1. 监控中心 (`/`, `/index.html`)

**实时监控看板**
- 核心指标：活跃 Agent、今日任务、Token 消耗
- 任务分布图：已完成/进行中/待处理
- Agent 状态：12 个部门实时状态
- 最近活动：任务执行记录
- 绩效表格：各部门任务统计
- 新闻聚合：AI/行业/公司新闻
- 流转审计：任务状态变更历史

### 2. 任务管理

**任务看板 (`/kanban.html`)**
- 看板视图：待处理/进行中/已完成三列
- Agent 筛选：按部门查看任务
- 优先级显示：高优先级标记
- 点击流转：快速切换状态

**任务管理 (`/tasks.html`)**
- 任务 CRUD：创建/查看/编辑/删除
- 状态管理：待处理→进行中→已完成
- 任务干预：暂停/恢复/转交/优先级调整
- 搜索筛选：按状态/关键词筛选

### 3. 组织管理

**组织架构 (`/org.html`)**
- 组织架构图：可视化展示 12 部门关系
- 部门职责：各部门核心职责和流程
- 协作关系：部门间协作矩阵

**行事准则 (`/conduct.html`)**
- 核心价值观：目标导向、协作共赢、创新驱动、合规经营
- 沟通准则：及时响应、透明沟通、尊重他人
- 决策准则：数据驱动、快速迭代、权责对等
- 执行准则：结果导向、主动推进、质量第一
- 协作准则：边界清晰、补位意识、知识分享
- 审批权限矩阵：各级审批权限
- 会议制度：站会/周例会/月度复盘

### 4. 会议管理 (`/meeting.html`)

**会议功能**
- 会议创建：主题/类型/时长/参会人员
- 会议类型：站会、评审会、战略会、培训会、1 对 1
- 快捷调度：快速站会、评审会议、战略会议
- 会议状态：待开会/进行中/已完成
- 会议议程：议程管理

### 5. 政策制度 (`/policies.html`)

**政策分类**
- 运营规范：日常运营流程
- 财务制度：报销/预算/核算
- 技术规范：代码审查/技术标准
- 人力资源：考勤/休假/绩效
- 合规风控：数据合规/风险控制

### 6. 技能管理 (`/skills.html`)

**技能配置**
- 技能列表：所有 Agent 技能
- 启用/禁用：开关控制
- 新增技能：自定义技能
- 默认技能：任务分配、质量检查、文档审查、战略会议、预算规划

### 7. 模型配置 (`/models.html`)

**AI 模型管理**
- 模型列表：配置多个 AI 模型
- 热切换：修改后即时生效
- 多提供商：阿里百炼、智谱 AI、OpenAI、Anthropic、MiniMax

### 8. 审核机制

**任务审核**
- 创建审核：审批/拒绝/批注
- 审核流程：审核通过后自动完成任务
- 审核记录：历史审核查询

### 9. 流转审计

**操作审计**
- 状态变更：记录每次状态变化
- 操作日志：谁在何时做了什么
- 原因记录：变更原因说明

### 10. 实时推送

**WebSocket 实时通知**
- 任务创建通知
- 任务状态变更
- 任务干预通知
- 新闻推送

## 技术架构

### 后端
- Python HTTP Server
- SQLite 数据库
- WebSocket 实时推送

### 前端
- TailwindCSS 样式
- Chart.js 图表
- 原生 JavaScript

### 数据表
- `tasks` - 任务表
- `reviews` - 审核表
- `audit_logs` - 流转审计
- `models` - 模型配置
- `skills` - 技能管理
- `news` - 新闻聚合
- `activities` - 活动日志
- `metrics` - 性能指标

## API 接口

### 任务相关
```
GET    /api/tasks              # 获取任务列表
POST   /api/tasks              # 创建任务
POST   /api/tasks/{id}/status  # 更新状态
POST   /api/tasks/{id}/intervene  # 任务干预
DELETE /api/tasks/{id}         # 删除任务
```

### 审核相关
```
GET    /api/reviews            # 获取审核列表
POST   /api/reviews            # 创建审核
POST   /api/reviews/{id}/status # 更新审核
```

### 审计相关
```
GET    /api/audit-logs         # 获取审计日志
```

### 配置相关
```
GET    /api/models             # 获取模型列表
POST   /api/models/{id}        # 更新模型
GET    /api/skills             # 获取技能列表
POST   /api/skills/{id}        # 更新技能
GET    /api/news               # 获取新闻
POST   /api/news               # 创建新闻
```
