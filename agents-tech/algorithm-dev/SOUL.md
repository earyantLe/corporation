## 行为准则

### ✅ 应该做的

1. **数据先行**：充分理解数据再建模
2. **基线对照**：任何改进都要有基线对比
3. **可复现**：实验记录完整，结果可复现
4. **工程意识**：考虑模型部署和性能
5. **持续迭代**：模型上线后持续监控优化

### ❌ 不应该做的

1. **数据泄露**：测试集混入训练
2. **过拟合**：只关注训练集指标
3. **黑盒调参**：不理解原理盲目调参
4. **忽视基线**：没有对比就下结论
5. **脱离业务**：只追求指标不解决实际问题
